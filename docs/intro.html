Rappel Mathématique¶

L’analyse marginale et les approximations¶

Astuce de l’approximation:

  • Fonctions généralement compliquées, fonctions linéaires simples…

  • Localement, on peut faire une approximation de toute fonction « lisse » à \(x_0\), pour \(x\) près de \(x_0\):

    \[\begin{aligned} f(x) \simeq f(x_0) + \alpha (x-x_0) + ... \end{aligned}\]

Exercice A: Montrer graphiquement une approximation de premier ordre.

Lien avec dérivée Pour \(x\) près de \(x_0\)

\[ \begin{align}\begin{aligned}\begin{split} \begin{aligned} &f(x) \simeq f(x_0) + \alpha (x-x_0) \\ \\ \iff & f(x) -f(x_0) \simeq \alpha (x-x_0)\\\\ \iff & \alpha \simeq \frac{f(x) -f(x_0)}{x-x_0} \simeq\; f'(x_0) \quad \text{par définition}\end{aligned}\end{split}\\Alors\end{aligned}\end{align} \]
\[\begin{split}\begin{aligned} &f(x) \simeq f(x_0) + f'(x_0) (x-x_0) \quad \text{ ou }\\ \\ &f(x) - f(x_0) \simeq f'(x_0) (x-x_0) \quad \text{ ou } \\ \\ &\Delta f \simeq f'(x_0) \Delta x\end{aligned}\end{split}\]

Les dérivées de fonctions¶

Avec des constantes

  • \(f(x) = b + ax\): \(f'(x) = a\)

  • \(f(x) = \log x\): \(f'(x) = \frac{1}{x}\)

  • \(f(x) = e^{ax}\): \(f'(x) = ae^{ax}\)

  • \(f(x) = x^a\): \(f'(x) = a x^{a-1}\)

Avec des fonctions

  • \(f(x) = a(x)b(x)\), \(f'(x) = a'(x)b(x) + a(x)b'(x)\)

  • \(f(x) = \frac{a(x)}{b(x)}\), \(f'(x) = \frac{a'(x)b(x) - a(x)b'(x)}{b(x)^2}\)

  • \(f(x) = a(b(x))\), \(f'(x) = a'(b(x))b'(x)\)

  • \(f(x) = a(x) + b(x)\), \(f'(x) = a'(x) + b'(x)\).

Exercice B: Trouvez les dérivées de : \(f(x)=\sqrt{x},\frac{x}{1+x},\frac{1}{2}x^2 + 2x-10,(1+\frac{x}{2})^2\).

** Approximations d’ordres supérieurs **

On peut pousser l’astuce plus loin

  • Polynome d’ordre 2 assez simple…

  • Alors, on approxime par deuxième ordre

  • Polynome d’ordre \(k\) …

  L’astuce porte le nom d’approximation de Taylor. Connectée aux dérivées d’ordres supérieurs d’une fonction:

\[ \begin{align}\begin{aligned} \begin{aligned} f(x) = f(x_0) + f'(x_0)(x-x_0) +\frac{1}{2}f''(x_0)(x-x_0)^2 + \ldots \end{aligned}\\On dénote :math:`f'(x), f''(x)` ou :math:`\frac{d f}{d x},\frac{d}{d x}(\frac{d f}{d x}) = \frac{d^2 f}{d x^2}`.\end{aligned}\end{align} \]

Concavité et Convexité des fonctions

Une fonction est convexe si pour tous points \((x_1,x_2)\) et tout \(\lambda>0,0<\lambda<1\):

\[\begin{aligned} f(\lambda x_1 + (1-\lambda) x_2) \leq \lambda f(x_1) + (1-\lambda)f(x_2)\end{aligned}\]

et concave si l’inverse est vrai. On dit strictement si les inégalités sont strictes.

Approximation et maximisation (sans contrainte)

Considérons l’approximation de premier ordre:

\[\begin{aligned} f(x_0+\Delta x) \simeq f(x_0)+ f'(x_0)\Delta x \end{aligned}\]

Observons que:

  • Si \(f'(x_0)>0\) un petit changement \(\Delta x>0\) augmente \(f\)

  • Si \(f'(x_0) <0\) un petit changement \(\Delta x <0\) augmente \(f\)

  • Si \(x_0\) est la solution de \(\max_x f(x)\), il faut que \(f'(x_0) =0\)

Maximisation sans contrainte

Considérons l’approximation de deuxième ordre:

\[\begin{aligned} f(x_0+\Delta x) \simeq f(x_0) + f'(x_0)\Delta x +\frac{1}{2}f''(x_0)\Delta x ^2 \end{aligned}\]

Pour un maximum (local), il faut que \(f'(x_0)=0\) (CPO) et \(f''(x_0)<0\) (CDO). Observons que:

  • Si \(f'(x_0) = 0\) mais \(f''(x_0)>0\), alors \(f(x_0+\Delta x) > f(x_0)\).

  • f’(x) doit être positif quand \(\Delta x <0\) et négatif quand \(\Delta x>0\).

Exercice C: Montrer graphiquement l’optimum d’une fonction concave.

Théorème de l’enveloppe¶

Considérons la fonction \(f(x,p)\) où \(p\) est hors de contrôle (exogène). On dénote:

\[V(p) = \max_x f(x,p) , \quad x^*(p) = \arg \max_x f(x,p)\]

On a que \(V(p) = f(x^*(p),p)\).

Ainsi,

\[V'(p) = f'_x(x^*(p),p)x^{*'}(p) + f'_p(x^*(p),p)\]

Or, puisque \(x^*(p)\) maximise \(f(x,p)\), \(f'_x(x^*(p),p) = 0\) par définition. Ainsi, \(V'(p) = f'_p(x^*(p),p)\).

Exercice D: Démontrer la valeur de cette approximation pour la fonction \(V(p) = (10 - p\frac{x}{2})x\).

Maximisation avec contrainte

Considérons le problème:

\[\begin{aligned} \max_x \{f(x):x \leq x_0\}\end{aligned}\]

Un maximum est atteint si \(f'(x)=0\) (nécessaire). Or si \(f'(x)>0\) pour tout \(x \leq x_0\), le maximum contraint est \(x_0\). Relacher la contrainte marginalement de \(x_0\) fait gagner \(f'(x_0)\Delta x\).

Plusieurs variables

Supposons la fonction \(f(x,y)\). La dérivée partielle se fait en gardant fixe (ou exogène) les autres variables: \(f'_x(x,y) = \frac{\partial f(x,y)}{\partial x}\).

La différentielle totale

Les combinaisons de \(x,y\) tels que \(f(x,y) = \overline{f}\)) peuvent être trouvées par solution directe, \(y=g(x,\overline{f})\). Mais on peut décrire ces combinaisons plus facilement en utilisant la différentielle totale (une approximation linéaire):

On peut décrire la forme de cette fonction par:

\[\begin{aligned} df(x,y) = f'_x(x,y)dx + f'_y(x,y)dy\end{aligned}\]

Si on pose \(df(x,y)=0\),

\[\begin{aligned} \frac{dy}{dx}\Bigr|_{df=0} = -\frac{f'_x(x,y)}{f'_y(x,y)}\end{aligned}\]

Exercice E: Trouvez \(\frac{dy}{dx}\Bigr|_{df=0}\) par différentielle totale pour \(f(x,y)=\log(xy)\).

Homogénéité des fonctions¶

Une fonction est homogène de degré \(r\) si pour tout \(\lambda>0\),

\[\begin{aligned} f(\lambda x_1, \lambda x_2, ... \lambda x_n) = \lambda^r f(x_1,x_2,...,x_n)\end{aligned}\]

Si une fonction est homogène de degré \(r\), alors ceci est aussi vrai (théorème d’Euler):

\[\begin{aligned} r f(x_1,x_2,...,x_n) = \sum_{i=1}^n \frac{\partial f}{\partial x_i}x_i\end{aligned}\]

Exercice F: Trouvez le degré d’homogénéité de la fonction \(f(x,y)=x^\alpha y^\beta\) des deux façons.

Approximation et Maximum

\[\begin{split}\begin{aligned} f(x,y) \simeq f(x_0,y_0) + f'_x(x_0,y_0)(x-x_0) + f'_y(x_0,y_0)(y-y_0) \\ +\frac{1}{2}f''_{xx}(x_0,y_0)(x-x_0)^2 + \frac{1}{2}f''_{yy}(x_0,y_0)(y-y_0)^2 + \\ +f''_{xy}(x_0,y_0)(x-x_0)(y-y_0) \end{aligned}\end{split}\]

Condition pour un maximum:

  • Nécessaire: \(f'_x(x,y)=0, f'_y(x,y)=0\)

  • Suffisante: \(\frac{1}{2}f''_{xx}(x_0,y_0)(x-x_0)^2 + \frac{1}{2}f''_{yy}(x_0,y_0)(y-y_0)^2 +f''_{xy}(x_0,y_0)(x-x_0)(y-y_0)<0\)

Maximisation avec contrainte

Approche directe. Quand le problème prend la forme:

\[\begin{aligned} \max_{x,y} \{ f(x,y): g(x,y)=m\}\end{aligned}\]

Et qu’on peut inverser \(g(x,y)=m\) tel que \(y=q(x,m)\), alors la solution du problème contraint pour \(x\) est la même que celui de :

\[\begin{aligned} \max_{x} \{ f(x,q(x))\}\end{aligned}\]

La CPO est \(f'_x(x,q(x,m)) + f'_y(x,q(x,m))q'(x,m) = 0\). On peut résoudre pour \(x^*\) et utiliser \(y=q(x)\) pour trouver \(y^*\). Facile, mais quand plus de 2 variables?

Exercice G: Maximiser la fonction \(f(x,y) = \log x + \log y\) sous la contrainte \(x+y \le m\).

Le Lagrangien¶

La méthode de Lagrange consiste à résoudre pour \((x,y)\),

\[\begin{split}\begin{aligned} f'_x(x,y) - \lambda g'_x(x,y) = 0 \\ f'_y(x,y) - \lambda g'_y(x,y) = 0 \\ g(x,y) = m\end{aligned}\end{split}\]

Ce sont les CPO du lagrangien:

\[\begin{aligned} \max_{x,y,\lambda} f(x,y) - \lambda (g(x,y)-m)\end{aligned}\]

Exercice H: Maximiser la fonction \(f(x,y) = \log x + \log y\) sous la contrainte \(x+y \le m\) par la méthode du Lagrangien.

L’interprétation du multiplicateur

La valeur de \(\lambda\) n’est pas nécessaire pour résoudre les valeurs optimales de \(x\) et \(y\). Mais elle a une interprétation.

Par le théorème de l’enveloppe, si

\[V(m) = \max_{x,y,\lambda} f(x,y) - \lambda (g(x,y)-m)\]

alors \(V'(m) = \lambda\).

Exercice I: Démontrer dans le problème précédent qu’une augmentation marginale de \(m\) sur le maximum est bien égale à \(\lambda\).
